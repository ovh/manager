# Architecture

# General information

The monorepo is powered by:

  • Yarn (opens new window) — Fast, reliable, and secure dependency management.
  • Lerna (opens new window) — A tool for managing JavaScript projects with multiple packages.

Several workspaces (opens new window) are configured and you can find the list in the root package.json (opens new window) file.

{
  "workspaces": [
    "packages/components/*",
    "packages/manager/apps/*",
    "packages/manager/modules/*",
    "packages/manager/tools/*"
  ],
  …,
}

# Languages and Framework

The Manager application is using these following languages:

  • HTML
  • CSS (Less.js (opens new window), SCSS (opens new window))
  • JavaScript

AngularJS (opens new window) used to be a widelys used MVW Framework to build Single Page Applications (opens new window).

Information

We are exploring some new solutions to speed up the development workflow.
