# Introduction

Manager is the control panel built on top of the OVHcloud API and based on our UI Framework.

Visit ovh.com/manager to manage your products or visit our Gitter to send us feedback.

Follow @ovhcloud on Twitter for important announcements.

This project adheres to the Contributor Covenant code of conduct. By participating, you are expected to uphold this code. Please report unacceptable behavior to opensource@ovh.net.
