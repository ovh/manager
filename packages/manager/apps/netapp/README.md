# manager-netapp

## Installation

Foremost, you should launch a global installation at the root folder of this repository:

```sh
yarn install
```

## Build

```sh
# Build in production mode
yarn start
```

## Development

If you want to contribute to the project, follow theses instructions:

Foremost, you should launch a global installation at the root folder of this repository:

```sh
yarn install
```

Then you just have to start the project in development mode. For this, two choices are possible according to your needs:

```sh
# Build the `manager-netapp` workspace and all the nested workspaces in development mode and watch only `manager-netapp` workspace
yarn start:dev
# Build and watch the `manager-netapp` workspace and all the nested workspaces in development mode
yarn start:watch
```
