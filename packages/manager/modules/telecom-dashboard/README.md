# telecom-dashboard

## Install

```sh
yarn add @ovh-ux/telecom-dashboard
```

## Usage

```js
import 'angular';
import telecomDashboard from '@ovh-ux/telecom-dashboard';

angular
  .module('myApp', [
    telecomDashboard,
  ]);
```
