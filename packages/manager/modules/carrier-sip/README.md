# manager-carrier-sip

> OVH Carrier SIP Trunk module.

## Install

```sh
yarn add @ovh-ux/manager-carrier-sip
```

## Usage

```js
import angular from 'angular';
import ovhManagerCarrierSip from '@ovh-ux/manager-carrier-sip';

angular.module('myApp', [ovhManagerCarrierSip]);
```
