# manager-email-domain

> OVH E-mail domain module.

## Install

```sh
yarn add @ovh-ux/manager-email-domain
```

## Usage

```js
import emailDomain from '@ovh-ux/manager-sharepoint';

angular.module('App', ['…', emailDomain, '…']);
```
