# manager-emailpro

> OVH E-mail Pro module.

## Install

```sh
pnpm install @ovh-ux/manager-emailpro
```

## Usage

```js
import emailpro from '@ovh-ux/manager-sharepoint';

angular.module('App', ['…', emailpro, '…']);
```
