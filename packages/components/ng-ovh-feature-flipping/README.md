## Install

```sh
$ yarn add @ovh-ux/ng-ovh-feature-flipping
```

## Usage

```js
import angular from 'angular';
import ngOvhFeatureFlipping from '@ovh-ux/ng-ovh-feature-flipping';

angular.module('myModule', [ngOvhFeatureFlipping]);
```

## Test

```sh
$ yarn test
```
