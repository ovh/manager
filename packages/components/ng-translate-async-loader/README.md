## Install

```sh
$ yarn add @ovh-ux/ng-translate-async-loader
```

## Usage

```js
import angular from 'angular';
import ngTranslateAsyncLoader from '@ovh-ux/ng-translate-async-loader';

angular.module('myApp', [ngTranslateAsyncLoader]);
```

## Test

```sh
$ yarn test
```
