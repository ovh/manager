# Onboarding Layout

## Overview

The `OnboardingLayout` component is a comprehensive, responsive layout designed to create engaging onboarding experiences for OVHcloud applications. It provides a structured template that combines a prominent header section with an optional grid of content cards, making it ideal for welcoming users and guiding them through new features or services.

## Recommandation

For the the `description` property, please be sure to wrap the component with a paragraph or Text component.
