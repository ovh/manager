The **Text** component is a wrapper for OdsText with added capability to handle IAM behaviour to displays a text if the user has access to the data, otherwise display a message with a tooltip for additional context. It is designed for interactive usage, with mandatory URN and IAM actions properties to associate the text with external resources.

The component supports internationalization (i18n).

</StorybookIdentityCard>

<StorybookHeading label="Anatomy" level={2} />

1. **ODS Text**: it is restricited to the "span" preset; it serves as the trigger for the tooltip
2. **ODS Tooltip**: it displays contextual information about the text

<StorybookHeading label="Usage" level={2} />

Use the **Text** to display the information requiring IAM check to display the text.

<StorybookHeading label="Placement" level={2} />

The **Text** component is typically used inline within a table, a paragraph, or alongside other UI elements where additional explanation is necessary.

<StorybookHeading label="Behavior" level={2} />

**States**

- **Default**:
  - The text is visible, and the tooltip is hidden by default
- **Hover and focus**:
  - All inherited from the ODS Tooltip component

**Interactions**
Hovering or focusing over the text displays the tooltip.

**URN**: links the text to an external resource.

**IAM actions**: related to IAM policies; One or more **actions** allowed or excepted by an IAM policy.

<StorybookHeading label="Variation" level={2} />

**Internationalization (i18n)**

The **Manage Text** component supports the following languages: English, German, Spanish, French (default), Italian, Dutch, Polish and Portuguese.
